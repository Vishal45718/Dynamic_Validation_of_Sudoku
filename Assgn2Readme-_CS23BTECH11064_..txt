Sudoku Validator - README


This program checks Sudoku solutions using multi-threading techniques to improve performance, especially for larger Sudoku grids. It checks the validity of rows, columns, and subgrids, using approaches like test and set,compare and swap ,,and bounded compare and swap.


input.txt: Input file containing the Sudoku grid.
output.txt: Output file that contains the final results and execution time.


→Compiled using C++ compiler.
